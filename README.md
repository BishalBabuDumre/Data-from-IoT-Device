# Data-from-IoT-Device
This repository has two codes. The first code is tempIrr.py. This code helps to get information about temperature and irradiance data obtained by a Meteorological center. The instrument that records temperature and irradiance has information about other relevant topics as well in a CSV file after the measurement in a remote device. We use tempIrr.py code to extract only necessary information from those CSV files and plot Temp and Irradiance VS Time of the Day plots. Another code emailGeneration.py helps to send those two generated plots to two different concerned people every other day via email feature. One sample CSV file has been attached.
